# Sammanfattning

Efter [bryning](/knowledge/techniques/bryning-och-fransning.md) sitter brun **fond** kvar i
pannan — koncentrerad smak. Avglasering löser upp den med vätska till grunden för en sås.

# Så gör du

1. Häll bort överflödigt fett, behåll fonden.
2. Häll i vätska (vin, buljong, vatten) i den heta pannan.
3. Skrapa loss fonden med slev medan vätskan sjuder.
4. [Reducera](/knowledge/techniques/reducera-sas.md) och smaka av; runda gärna med smör.

# Varför det fungerar

Fonden är vattenlösliga Maillard-produkter (se
[Maillardreaktionen](/knowledge/principles/maillardreaktionen.md)) som återgår i såsen.

# Vanliga fel

* Bränd (svart) fond → [beskt](/knowledge/troubleshooting/beskt.md); börja om.

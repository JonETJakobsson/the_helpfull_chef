# Sammanfattning

Vatten kokar vid 100 °C (vid havsnivå). **Sjudning** (små bubblor, ~85–95 °C) är
skonsammare än full kokning. När vätska avdunstar **koncentreras smak** och såsen
tjocknar — det kallas reduktion.

# Nyckelfakta

* Full kokning river sönder ömtåliga råvaror (fisk, ägg) och kan göra kött segt — sjud i stället.
* **Reduktion** koncentrerar både smak och salt — salta därför i slutet.
* Lock behåller vätska (bra för att koka mjukt), utan lock avdunstar den (bra för att reducera).
* Ju större yta, desto snabbare avdunstning — använd vid gryta för att reducera.

# Sammanfattning

Stärkelse (från mjöl, potatis, majs, ris) består av korn som **sväller och binder vatten**
när de värms i vätska — kallas gelatinisering. Det är mekanismen bakom redda såser,
soppor och grytor.

# Nyckelfakta

* Gelatinisering sker vanligen vid **ca 60–80 °C** beroende på stärkelsekälla.
* **Roux** (mjöl + fett) och **redning/slurry** (stärkelse + kall vätska) fördelar
  stärkelsen jämnt så den inte klumpar.
* Majsstärkelse och potatismjöl ger klar, blank sås; vetemjöl ger matt, ogenomskinlig.
* För mycket omrörning efter förtjockning (särskilt potatismjöl) kan **tunna ut** såsen igen.

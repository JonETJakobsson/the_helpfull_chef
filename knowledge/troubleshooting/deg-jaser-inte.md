# Symptom

Bröddeg höjer sig inte, eller bakverk blir platt och kompakt.

# Vanliga orsaker (jäst)

* **Gammal/död jäst** eller jäst dödad av för varm vätska (>50 °C).
* För kallt i rummet → mycket långsam jäsning (inte nödvändigtvis fel).
* För mycket salt eller socker direkt på jästen.

# Vanliga orsaker (kemisk hävning)

* Gammalt **bakpulver/bikarbonat**, eller bikarbonat utan syra i smeten.
* Smeten stått för länge före gräddning.

# Åtgärder

* Ge jästdeg **mer tid** på varmare plats; testa jästen i ljummet sockervatten.
* Kontrollera datum på jäst/bakpulver; byt vid tveksamhet.

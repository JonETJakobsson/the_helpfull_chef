# Calendars

Genererade `.ics`-kalenderfiler från veckomenyer, en fil per vecka
(`<år>-w<vecka>.ics`). Importera filen i din delade familjekalender så ser hela
familjen veckans middagar.

* [2026-w28.ics](2026-w28.ics) - Grillhelg fredag–söndag (10–12 juli 2026).
* [2026-w29.ics](2026-w29.ics) - Demo: middagar för vecka 29 (13–17 juli 2026).

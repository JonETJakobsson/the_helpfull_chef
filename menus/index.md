# Weekly Menus

Planerade veckomenyer, en fil per vecka (`<år>-w<vecka>.md`, t.ex. `2026-w28.md`).
Varje meny innehåller de valda vardagsmiddagarna samt de veckospecifika inputs som
användaren rapporterade vid planeringen (sug, råvaror att använda upp, sådant som redan
finns hemma).

* [2026-w28.md](2026-w28.md) - Grillhelg fredag–söndag (10–12 juli 2026).
* [2026-w29.md](2026-w29.md) - Demo: vardagsmiddagar för vecka 29 (13–17 juli 2026).

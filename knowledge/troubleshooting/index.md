# Felsökning

Symptom → orsak → åtgärd. Detta är motorn i feedback-driven receptförbättring: när
användaren säger "det blev X" slår agenten upp rätt post och föreslår konkreta justeringar.
Concept-typ: `Troubleshooting`.

| Concept | Symptom |
|---------|---------|
| [För salt](for-salt.md) | Rätten blev för salt — hur den räddas. |
| [Fadd / tunn smak](for-blaskigt.md) | Platt smak: salt, syra, umami saknas. |
| [Brusten sås](brusten-sas.md) | Emulsion som skiktat sig — laga och förebygg. |
| [Torrt / segt kött](torrt-kott.md) | Övertillagat eller fel styckning. |
| [Klumpig sås](klumpig-sas.md) | Klumpar vid redning — sila och förebygg. |
| [Vidbränt](brand-botten.md) | Bränd botten — rädda smaken. |
| [Deg jäser inte](deg-jaser-inte.md) | Jäst, temperatur, tid. |
| [Seg / hård deg](seg-deg.md) | Överarbetad gluten. |
| [För fet / oljig rätt](for-fet-ratt.md) | Skära av fett, balansera. |
| [Beska](beskt.md) | Bränd vitlök, överreducering, brända kryddor. |
| [För sött](for-sott.md) | Balansera med syra och salt. |
| [Vattnig gryta / röra](vattnig-gryta.md) | Reducera eller reda. |

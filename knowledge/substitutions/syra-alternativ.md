# När det behövs

Slut på den syra receptet anger, eller vill anpassa smakprofil.

# Alternativ (ungefärligt)

| I stället för | Använd | Notering |
|---------------|--------|----------|
| Citronsaft | Lika mängd vitvinsvinäger eller lime | Vinäger vassare, ingen citrusarom |
| Vitvinsvinäger | Citronsaft eller äppelcidervinäger | Justera efter smak |
| Vitt vin (i sås) | Buljong + skvätt vinäger/citron | Ger syra utan alkohol |
| Rödvinsvinäger | Balsamico (sötare) eller rödvin | Balsamico även sötma |
| Färsk citronsaft | Torkat citronskal + lite vinäger | Nödlösning |

# The Helpful Chef — Knowledge Bundle

Food-planning knowledge base. Everything here is authored and maintained by the
**The Helpful Chef** agent together with the user.

# Recipes

* [recipes/](recipes/) - Reusable recipes, one file per dish.

# Ingredients

* [ingredients/](ingredients/) - Canonical ingredient catalogue used to normalise
  names and build shopping lists.

# Household (private)

* [household/](household/) - Family members, per-member ratings and meal history.

# Outputs

* [menus/](menus/) - Planned weekly menus (5 weekday dinners each).
* [shopping-lists/](shopping-lists/) - Shopping lists generated from menus.

# Recipes

One file per dish. Each recipe groups its ingredients by component (sås, stek,
tillbehör, salladsmix …) and ends with an ordered timeline so every part is ready
at the same time.

* [Köttbullar med gräddsås och potatismos](kottbullar-med-graddsas-och-potatismos.md) - Klassiska köttbullar med krämig gräddsås och potatismos.
* [Ugnsbakad lax med citronsås och sparris](ugnsbakad-lax-med-citronsas-och-sparris.md) - Snabb vardagslax i ugn med len citronsås.
* [Ugnspannkaka med bacon](ugnspannkaka-med-bacon.md) - Klassisk matig ugnspannkaka med knaperstekt bacon.

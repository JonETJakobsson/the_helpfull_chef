# När det behövs

Slut på grädde, eller vill ha lättare/mjölkfritt alternativ i såser och soppor.

# Alternativ

| I stället för grädde | Använd | Notering |
|----------------------|--------|----------|
| Matlagningsgrädde | Crème fraîche eller gräddfil | Fylligt; tillsätt sent, kan skära vid kokning. |
| Matlagningsgrädde | Mjölk + smör (ca 2 dl mjölk + 2 msk smör) | Tunnare; [reda](/knowledge/techniques/reda-sas.md) vid behov. |
| Matlagningsgrädde | Kokosgrädde | Mjölkfritt; ger kokossmak. |
| Vispgrädde (till vispning) | Kokosgrädde (kyld) eller vispbar växtgrädde | Mjölkfria alternativ. |
| Grädde i soppa | Mixad kokt potatis/cashew | Ger krämighet utan mejeri. |

# Att tänka på

* Magra alternativ **skär sig** lättare — undvik full kokning, tillsätt sent.
* Syrade produkter (crème fraîche) tål värme sämre än grädde.

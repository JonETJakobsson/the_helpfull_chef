# Fråga först: vad gör ägget?

Ägg kan **binda**, **häva** (luft) och ge **fukt/fett**. Välj ersättning efter funktion.

# Alternativ (per ägg)

| Funktion | Ersättning |
|----------|------------|
| Binda | 1 msk malda linfrön + 3 msk vatten (svälls 10 min); eller 3 msk aquafaba |
| Binda/fukt | 65 g mosad banan eller äppelmos (ger smak/sötma) |
| Häva | 3 msk aquafaba vispad; eller 1 tsk bakpulver + 1 msk olja + 1 msk vatten |
| Fukt | 3 msk yoghurt eller växtyoghurt |

# Att tänka på

* Ingen ersättning gör allt — i recept med **många** ägg (t.ex. sockerkaka som lyfts av
  ägg) blir resultatet svårare att matcha.
* Häv-behov kopplar till [hävning (bakpulver/bikarbonat)](/knowledge/principles/havning-bakpulver-bikarbonat.md).

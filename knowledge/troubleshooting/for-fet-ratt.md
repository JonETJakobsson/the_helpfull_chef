# Symptom

Rätten känns tung och oljig, eller olja lägger sig på ytan.

# Vanliga orsaker

* För mycket tillsatt fett/olja.
* Fett från kött/korv som frigjorts i grytan.
* Bristande balans mot syra.

# Åtgärder

* **Skumma av** ytfett med sked, eller dra en pappershushållsbit över ytan.
* Kyl grytan så stelnar fettet på ytan och kan lyftas av.
* Tillsätt [syra](/knowledge/principles/syra-och-smakbalans.md) (citron, vinäger, tomat) som
  skär genom fettet.
* Servera med något som suger upp (bröd, ris, potatis).

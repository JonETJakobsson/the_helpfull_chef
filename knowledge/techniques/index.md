# Tekniker

Återanvändbara metoder — det praktiska *hur:et*. Concept-typ: `Technique`.

| Concept | Täcker |
|---------|--------|
| [Bräsering](brasering.md) | Långsam tillagning i vätska för sega köttdetaljer. |
| [Reducera sås](reducera-sas.md) | Koka in vätska för att koncentrera smak och tjocklek. |
| [Emulgera sås](emulgera-sas.md) | Bygga emulgerade såser (hollandaise, vinägrett, aioli). |
| [Reda sås](reda-sas.md) | Förtjocka med roux, redning eller beurre manié. |
| [Bryning & fräsning](bryning-och-fransning.md) | Steka på hög värme för yta och smak. |
| [Blanchering](blanchering.md) | Snabbkoka och kyla för färg och textur. |
| [Pochering](pochering.md) | Skonsam tillagning i het vätska under kokpunkten. |
| [Karamellisera lök](karamellisera-lok.md) | Långsamt bryna lök till sötma. |
| [Vila kött](vila-kott.md) | Låta kött vila så saften fördelas. |
| [Torrsaltning](torrsaltning.md) | Salta i förväg för smak och saftighet. |
| [Marinering](marinering.md) | Marinader med syra, enzym och salt. |
| [Avglasering (deglaze)](avglasering.md) | Lösa upp fond med vätska. |
| [Smaka av & balansera](smaka-av-och-balansera.md) | Justera salt, syra, fett, sötma och hetta. |
| [Knåda deg](knada-deg.md) | Knådning för glutenstruktur. |
| [Vika & vispa](vika-och-vispa.md) | Vika och vispa för luftighet. |
| [Temperera](temperera.md) | Temperera ägg/choklad för att undvika skärning/stelning. |
